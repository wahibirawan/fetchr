# Fetchr - Image Downloader & Bulk Grabber

Fetchr is a premium browser extension designed to inspect, preview, and batch-download high-resolution raster images (JPG, PNG, WebP) from any website. It acts as an ultimate asset puller tool for designers, developers, and general users seeking a clean and efficient workflow.

## Features

- **Deep Image Detection**: Extracts background images, absolute/relative sources, and handles modern formats like WebP and AVIF.
- **Raster Exclusive**: Filters out SVG and non-raster files to focus strictly on standard image assets.
- **Sorting Options**: Sort collected images by DOM appearance order, size (ascending or descending), or file type.
- **Quick Actions**: Easily copy individual images to the clipboard or download them to your local device.
- **Bulk Download**: Save all collected images at once with an organized sequential naming convention.
- **Premium UI**: Features a modern, tactile, and glassmorphic interface built for speed and visual clarity.

## Installation

1. Download the extension package: [Fetchr.zip](./Fetchr.zip)
2. Extract the ZIP file to a folder on your computer.
3. Open your Chromium-based browser (Chrome, Edge, Brave, etc.).
4. Navigate to the extensions page (`chrome://extensions` or `edge://extensions`).
5. Enable "Developer mode" in the top right corner.
6. Click "Load unpacked" and select the folder where you extracted the extension.

## Usage

1. Navigate to the webpage you want to extract images from.
2. Click the Fetchr extension icon in your browser toolbar.
3. Wait a moment for the extension to scan the page.
4. Use the dropdown filter to sort the images.
5. Click "Copy" to copy an image to the clipboard, "Save" to download an individual image, or "Download All" to grab everything at once.
